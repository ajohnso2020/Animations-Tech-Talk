Introduction

CSS Animations are like transitions in that they display the change of a CSS property from one state to another.

However, animations are most powerful because they involve many more in-between states and they offer more control over how those states are actually animated.

Building Blocks

Keyframes: describe the stages and style of the animation.

Animation Properties: these keyframes are assigned to a specific CSS property and define how they are animated.

      @keyframes bounceIn {
    0% {
      transform: scale(0.1);
      opacity: 0;
    }
    60% {
      transform: scale(1.2);
      opacity: 1;
    }
    100% {
      transform: scale(1);
    }
  }

Keyframe Components

  • The animation name like "slideIn"
  • The stages of the animation in percentages like 0%, 50% and 100%
  • The specific CSS properties eligible for animation like font-size, height and left-margin

Animation Properties

  • Animation-name -> is the name defined keyframes and that describes the animation like bounceIn and fallDown.
  • Animation-duration -> is the duration of the animation defined in seconds or milliseconds.
      div {
    animation-duration: 2s;
    animation-name: bounceIn;
  }

Additional Animation Properties

animation-timing-function -> describes the pace of the animation like "ease-in" and "linear".

      animation-timing-function: ease-in-out;

animation-delay -> specifies when it starts in seconds or milliseconds.

animation-iteration-count -> specifies how times the animation plays from once to infinite times.

        animation-iteration-count: 2;

animation-direction -> specifies the direction it should play like in forward or reverse.

animation-fill-mode -> it describes how the animation is effected before and after it is played.

animation-play-state -> specifies when it plays or when it is paused.

        .div:hover {
    animation-play-state: paused;
  }
